hexo-asset-image 图片测试

发表于 2016-06-26   |   分类于 Hexo   |  

本文是测试hexo图片功能的，这里使用了hexo-asset-image插件，官网,

使用方法

一、 在配置文件_config.yml中把，post_asset_folder: true设为 true
二、 把图片复制到与文章文件同名的文件夹下，然后在文章文件中引用图片即可，这样在我们更新文件到github是就会自动把图片上传。

这里的例子是image-test.md,image-test文件夹下的test.jpg是我们测试用的图片。

详细请查看 github官方 和 简书
