用GitHub+hexo搭建博客成功 ShanFeng博客

发表于 2016-06-26   |   分类于 GitHub   |  

WordPress 真的不想用了，不是不好用而是被github这个强大而又免费的服务所引诱的。如果你只是用博客记录一些自己的日志的话，用GitHub绝对不会让你失望，下面列出github+hexo搭建博客的好处。

使用github 的基础就是会使用git,git官方文档

github+hexo的好处

  1. github、hexo，都是开源免费的项目，github是免费的服务，hexo是免费的工具，使用两者的原因是 可能你用wordpress自己搭建的网站一年以后就会找不到，可能因为你没有为你的主机续费也可能是其他原因，而使用github+hexo可能20年后你的博客依然存在，因为GitHub就是那么强大。
  2. hexo 生成的是纯静态的文件，所以网页打开的速度不会很慢。
  3. hexo 可能要比wordpress 对markdown支持的更好。
