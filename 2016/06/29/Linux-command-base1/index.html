mkdir 命令

用途

创建一个或多个目录，可以单个创建也可以嵌套创建。

参数

mkdir 有三个参数，-e-m-p
-e 以加密继承创建目录
-m 设置新创建目录的许可位，其值有变量 mode 指定。mode 变量的值与 chmod 命令的 mode 参数的值一样，或者以符号形式，或者以数字形式。
-p 创建嵌套的目录（个人简单理解）
