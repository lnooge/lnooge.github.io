添加Nginx到yum源

安装nginx

-v代表显示安装过程
-y代表安装过程中要选择y or n 时自动全部y

配置文件说明

  1. 配置目录
    网站文件存放默认目录
    /usr/share/nginx/html
    网站默认站点配置
    /etc/nginx/conf.d/default.conf
    自定义Nginx站点配置文件目录
    /etc/nginx/conf.d/
    nginx全局配置
    /etc/nginx/nginx.conf

  2. 启动目录
    /usr/sbin/nginx
    参数：

    1. -t 测试配置文件
2. -v 显示版本(show version and exit)
3. -V 显示版本和配置选项（show version and configure options then exit）
4. -T (test configuration, dump it and exit)
5. -q (suppress non-error messages during configuration testing)
6. -s signal 停止、退出、重启、重新加载配置文件(send signal to a master process: stop, quit, reopen, reload)
7. -p refix (set prefix path(default: /etc/nginx/))
8. -c filename (set configuration file (default: /etc/nginx/nginx.conf))
9. -g directives (set global directives out of configuration file)

  3. CentOS 开机启动Nginx
